**版本**: v1.2.0  
**作者**: 墨墨 (Mò) / whiteareas

## 📦 安装

### 水产市场
```bash
openclawmp install skill/@u-9e6ebb2ab773477594f5/config-center
```

### ClawHub
```bash
clawhub install openclaw-config-center
```

## 🎯 核心功能

1. **配置集中管理** - `~/.openclaw/config/` 模块化配置
2. **动态配置读取** - `config-loader.sh` 配置加载器
3. **主配置自动融合** - `generate-main-config.sh`
4. **记忆自动同步** - `update-soul.sh`
5. **ClawRouter 成本优化** - 节省 74% API 成本

## 📊 效率提升

- 改配置时间：10 分钟 → 1 分钟 (**10 倍**)
- API 成本：$450/月 → $117/月 (**节省 74%**)
